# ReelBot
No more manual scrolling to watch reels, this bot filters out most viewed reels for a given profile

# Two main features

## 1. getTop

- Get recent 10 most viewed reels
```
/getTop [profileURL]
```

## 2. getAll

- Get recent over million viewed reels
```
/getAll [profileURL]
```
